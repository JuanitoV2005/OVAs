Introducción a Arrays en Java

Los arrays en Java son estructuras de datos que permiten almacenar múltiples valores del mismo tipo en una sola variable. Cada valor en el array se guarda en una posición determinada, llamada índice, que empieza desde 0.

Por ejemplo, un array de enteros puede almacenar una lista de números enteros:

int[] arr = {10, 22, 43, 84, 5};

O también de números tipo double:

double[] arr = {5.8, 22.8, 35.9, 47.1, 2};

En los siguientes apartados exploraremos cómo obtener su tamaño, acceder a elementos individuales y modificar su contenido.

Obtener la longitud de un array

Todos los arrays en Java tienen un atributo llamado length que indica cuántos elementos contiene el array. Esto es muy útil para recorrer el array sin salirnos de sus límites.

System.out.println(arr.length);

Si nuestro arreglo es {1, 2, 3}, la salida será:

3

Acceder al primer elemento

Podemos acceder a cualquier posición de un array usando su índice entre corchetes. El primer elemento está en la posición 0.

System.out.println(arr[0]);

Esto imprimirá el valor del primer elemento del arreglo, el cual puede cambiar dinámicamente según lo que contenga arr[0].

Modificar un elemento

También es posible asignar nuevos valores a posiciones específicas del array. Por ejemplo, podemos cambiar el valor en la posición 0 de la siguiente manera:

arr[0] = 9;

System.out.println(arr[0]);

En este caso, sin importar el valor anterior, el primer elemento pasará a ser 9.
